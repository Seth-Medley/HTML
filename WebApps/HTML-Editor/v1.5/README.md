# HTML Editor v1.5

## Changes from v1.4
- **Modularization**: The `function.js` file is now fully indexed and organized into logical blocks.
- **HTML5 Boilerplate**: Updated the main wrapper to follow modern HTML5 accessibility standards.
- **Full Separation**: Styles are completely offloaded to `CSS/styles.css`, and the HTML file contains zero inline scripting.

## Code Comparison
- **Lines of Code**: ~80+ lines across 3 files.
- **Difference**: This is the most significant architectural change. While the visual output remains similar to v1.0, the "under the hood" code is now professional-grade, scalable, and easy to maintain.
