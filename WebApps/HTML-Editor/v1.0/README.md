# HTML Editor v1.0

## Description
The initial prototype of the HTML Editor. Focuses on the basic "Hello World" of web editors: taking raw text and injecting it into an iframe.

## Changes & Versioning
- **Initial Build**: Established the dual-pane layout (Input vs. Result).
- **Core Function**: Introduced the `run()` function to update the iframe.

## Code Comparison
- **Lines of Code**: ~30 lines (HTML/JS combined).
- **Difference**: This version uses a very simple `document.getElementById().value` approach without any error handling or initialization logic.
