## How to Use
1. Navigate to the latest version folder (`/v1.5`).
2. Open `HTML Editor_v1.5.html` in any modern web browser.
3. Input HTML/CSS/JS in the editor and view the live result in the preview pane.

## Features
* **Real-time Preview**: See code changes instantly in the result pane.
* **Modular Design**: Clean separation of styles and logic for easy maintenance.
* **Lightweight**: Zero external dependencies (pre-v1.3) or minimal jQuery (v1.3+).
