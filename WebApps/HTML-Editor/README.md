# HTML Editor Project

A progressive web-based HTML editor that allows for real-time code execution and previewing. This project tracks the evolution from a simple text-area input to a modular, jQuery-powered development tool.

## Project Structure
The project is divided into versions to demonstrate development milestones:
- **v1.0 - v1.2**: Vanilla JavaScript foundations.
- **v1.3**: Introduction of jQuery.
- **v1.4 - v1.5**: Optimization, modularity, and UI refinements.

## How to Use
1. Clone the repository.
2. Navigate to the latest version (v1.5).
3. Open `HTML Editor_v1.5.html` in any modern browser.
4. Input HTML/CSS/JS in the editor and view the live result in the preview pane.

## Release Strategy
For each version listed below, a GitHub Release tag should be created. Each release includes the standalone HTML file and its associated JS/CSS assets.
