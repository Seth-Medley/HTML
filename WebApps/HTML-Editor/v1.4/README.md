# HTML Editor v1.4

## Changes from v1.3
- **Modern Standards**: Switched from `var` to `const` and `let` for better scoping.
- **UI Feedback**: Added an `alertTime` function to provide timed notifications to the user.
- **Auto-Update**: Improved the `run()` function to trigger more reliably on input change.

## Code Comparison
- **Lines of Code**: ~65 lines.
- **Difference**: Increase of ~20 lines. The focus here was on "User Experience" (UX) by adding feedback mechanisms and safer variable declarations.
