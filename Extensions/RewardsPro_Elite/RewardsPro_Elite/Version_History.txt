[VERSION 3.7] - THE "PURE STEALTH" BUILD (CURRENT)

[VERSION 3.6] - NATURAL LANGUAGE & LOGGING
--------------------------------------------------
* FEATURE: Replaced random word strings with a "Natural Language 
  Query Generator." Now forms human-like questions using a 
  200-word bank (Starters, Subjects, Actions, Contexts).
* FEATURE: Restored the "Mission Log" activity box to the 
  popup for real-time status monitoring.
* FEATURE: Integrated "Human Jitter" (random page scrolling) 
  and "Adaptive Typing Jitter" (variable ms delays per character).

  scraping logic to bypass Bing's anti-bot/shadow-dom measures.

  header monitoring to track PC/Mobile points.
* RESULT: Removed due to Bing's security headers and UI 
  instability. Features were stripped to maintain stealth.
