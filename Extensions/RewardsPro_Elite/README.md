# 💎 Rewards Pro: Elite v4.0 - Official Release

This is the stable v4.0 "Genesis" build, featuring the Ghost Engine architecture and high-fidelity natural language generation.

## 🛠️ How to Install (Fixes 'Invalid Package' Error)
Chrome often blocks direct `.crx` files from GitHub for security. Follow these steps for a guaranteed installation:

1. **Download**: Click the `RewardsPro_Elite_v4.0.zip` under **Assets** below.
2. **Extract**: Right-click the downloaded ZIP and select **Extract All**.
3. **Manage Extensions**: Open Chrome and go to `chrome://extensions`.
4. **Developer Mode**: Ensure the toggle in the top-right corner is **ON**.
5. **Install**: Click **Load unpacked** and select the folder you just extracted.
